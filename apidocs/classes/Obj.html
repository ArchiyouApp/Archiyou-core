_compileStyle

  • _compileStyle(style: any): ObjStyle

  • Compile different parameters into a ObjStyle instance we can have different levels of detail:

    • baseStyle: { color, opacity, size } which translates to all geometries (point, line, fill)
    • objStyle: { point: BaseStyle, fill: BaseStyle, line: BaseStyle } which we just copy directly over after checking NOTE: baseStyle params can also be directly set on the Obj: box.color = 'red'

    Parameters

    • style: any

    Returns ObjStyle
